Description: The transform.py python scripts reads superbowl.html file, extracts required html data
             using Beautifulsoup, it then uses the extracted html data and extracts the required data fields.


Execution Steps:
1. Place the superbowl.html file in same folder as transform.py script.
2. Run transform.py script.
   It will read and parse the html file and output result.csv
3. Once the execution of transform.py is completed, check result.csv for output.


List of Files:
1. transform.py
2. result.csv
3. README_2.txt


Required packages:
1. BeautifulSoup
